About Crypto Portfolio

A modern web application for tracking Bitcoin holdings and market prices, built with the latest web technologies.

Our Mission

Crypto Portfolio was created to provide users with a simple, elegant way to track their Bitcoin investments and stay updated with real-time market prices. Our platform offers:

  • Real-time Bitcoin price tracking in USD and INR
  • Portfolio value calculation based on current market prices
  • Historical price charts and market data
  • User account management with secure authentication
  • Responsive design that works on all devices

Technology Stack

This application is built using modern web technologies:

Frontend
  • HTML5 & CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • Chart.js for data visualization
Backend
  • PHP 8+
  • PHPMailer for email functionality
  • CoinGecko API integration
  • Session-based authentication
