Our prices are fetched directly from CoinGecko's API, which aggregates data from multiple exchanges. The prices are updated in real-time and are among the most accurate available. However, please note that cryptocurrency prices can be volatile and may vary slightly between different exchanges.

Yes, your portfolio data is stored securely. This is a demo application that uses browser localStorage for temporary storage. For production use, we would implement proper database storage with encryption. We never store your private keys or have access to your actual cryptocurrency wallets.

No, you can use the basic portfolio calculator without creating an account. However, registering allows you to save multiple portfolios, receive email notifications, and access additional features. Registration is free and only requires an email address and password.

Currently, we support Bitcoin price display in USD (United States Dollar) and INR (Indian Rupees). We chose these currencies to cater to our primary user base. Additional currency support may be added in future updates based on user feedback.

Bitcoin prices are updated automatically every 30 seconds when you're actively viewing the application. This ensures you have access to near real-time pricing data without overwhelming the API servers. You can also manually refresh the prices at any time.
